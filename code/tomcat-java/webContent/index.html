在浏览器的地址中输入url，url

1.在浏览器的地址中输入url，url 中包含服务器软件所在计算机的ip地址和所占用的端口号

2.浏览器会根据http协议，将url包装成请求，通过tcp协议发送给服务器软件

3.服务器软件根据http协议解析请求，并根据描述请求中的描述找到相应的资源

4.服务器软件将找到的资源按照http协议的要求，发回给浏览器
